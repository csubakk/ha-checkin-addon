# Fonts

Ez a mappa tartalmazza a PDF generáláshoz használt betűtípusokat.

## Tartalom

- `DejaVuSans.ttf`: Unicode-kompatibilis, nyílt forráskódú betűtípus, amely tökéletes PDF-ek generálásához többnyelvű karakterkészlettel.
